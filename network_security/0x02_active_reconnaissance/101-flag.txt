Active reconnaissance is the process of directly interacting with a target system to gather information such as open ports, services, technologies and vulnerabilities; it is important in cybersecurity because it allows attackers and defenders to understand the attack surface; Wappalyzer can be used for active reconnaissance to identify web technologies (server, frameworks, CMS and libraries) by analyzing a live website; DNS enumeration is the process of discovering DNS records and related assets such as subdomains, name servers and IP addresses; SMTP enumeration can be performed with command-line tools (e.g., nmap scripts or smtp-user-enum) to identify server capabilities and valid users; OS fingerprinting should be performed by sending network probes and analyzing responses (commonly with nmap -O) to infer the operating system; sqlmap is an automated SQL injection tool used to detect, enumerate and exploit SQL injection vulnerabilities in web applications.
